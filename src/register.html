Create an account

To create an account, follow the steps below:

  1. Enter your email address below.
  2. Click the Send verification code button.
  3. Check your email account for an email arriving shortly from "Microsoft on behalf of business.gov.au".
  4. Copy the code from your email and return to this page.
  5. Enter the verification code into the box that will appear below.
  6. Click the Verify code button.
  7. If this doesn't work, click the Send new code button and repeat steps 3 to 6.
  8. Fill in the rest of the details and select the Create button.
